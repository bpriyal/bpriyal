Model the Problem

Model option price \(V(S,t)\) using the PDE: \[ \frac{\partial V}{\partial t} + r S \frac{\partial V}{\partial S} + \frac{1}{2}\sigma^2 S^2 \frac{\partial^2 V}{\partial S^2} - r V = 0 \] Discretize using time steps \(\Delta t\) and stock steps \(\Delta S\).

Solution in Math (Explicit Euler)

\[ \frac{\partial V}{\partial t} \approx \frac{V_i^{j+1} - V_i^j}{\Delta t},\quad \frac{\partial^2 V}{\partial S^2} \approx \frac{V_{i+1}^j - 2V_i^j + V_{i-1}^j}{(\Delta S)^2} \]

Extrapolation (C++ Implementation)

  • Grid stored in std::vector<double> (current time slice)
  • Stability: \(\Delta t / (\Delta S)^2 < 1/2\). Otherwise, switch to Implicit Method solving a tridiagonal system
