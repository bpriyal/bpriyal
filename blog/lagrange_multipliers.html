Primer Ch. 8.1 (Lagrange Multipliers)
C++ Implementation: lagrange_optimizer.cpp

Model the Problem

Maximize function \(f(x,y)\) subject to constraint \(g(x,y)=c\). Define Lagrangian: \[ L(x,y,\lambda) = f(x,y) - \lambda (g(x,y)-c) \]

Solution in Math

Solve system:

\[ \frac{\partial L}{\partial x} = 0,\quad \frac{\partial L}{\partial y} = 0,\quad \frac{\partial L}{\partial \lambda} = 0 \]

Extrapolation (C++ Implementation)

  • Solve linear system \(Ax=b\) using covariance matrix and constraint vectors
